# Symfony-S5

### Cloner le projet

    git clone https://github.com/SamChrp/Symfony-S5.git
    cd Symfony-S5
    composer install ou composer update

### Créer la dbdd
    bin/console d:d:c

### Mettre à jour la bdd

    bin/console d:s:u -f

### Lancer les fixtures

    bin/console d:f:l

### lancement du serveur

    symfony server:start
